# Admin Bank - API Backend

Esta es la API del sistema bancario, construida para gestionar la persistencia de datos.

## Cómo ejecutar
1. Asegúrate de tener instalado [Node.js](https://nodejs.org/).
2. Instala las dependencias:
   npm install
3. Inicia el servidor de datos (json-server):
   npx json-server --watch db.json --port 3001
4. La API estará disponible en http://localhost:3001.
